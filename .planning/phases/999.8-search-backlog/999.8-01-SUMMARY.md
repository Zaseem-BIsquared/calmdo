## Decisions Made
- Client-side filtering in listFiltered — Convex doesn't support compound queries across multiple filter dimensions on different indexes; for expected data volume (< 10K tasks) this is efficient
- assigneeId accepts "me" and "unassigned" as special string values mapped to userId comparison and null check respectively
- Default search limit of 10 results, overridable via `limit` parameter

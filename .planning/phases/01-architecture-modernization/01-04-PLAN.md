Reduce each route file to a thin wrapper. Pattern:

```typescript
import { createFileRoute } from "@tanstack/react-router";
import { DashboardPage } from "@/features/dashboard";

export const Route = createFileRoute("/_app/_auth/dashboard/_layout/")({
  component: DashboardPage,
  beforeLoad: () => ({
    title: "Dashboard",
    headerTitle: "Dashboard",
    headerDescription: "Overview of your account.",
  }),
});
```

Apply to all 5 page route files:
- `_layout.index.tsx` -> imports DashboardPage from `@/features/dashboard`
- `_layout.checkout.tsx` -> imports CheckoutPage from `@/features/billing`
- `_layout.settings.index.tsx` -> imports SettingsPage from `@/features/settings`
- `_layout.settings.billing.tsx` -> imports BillingSettings from `@/features/billing`
- `_layout.username.tsx` -> imports UsernamePage from `@/features/onboarding`

For `-ui.navigation.tsx`: this file is used as a TanStack Router convention file (the `-ui.` prefix). Keep it but make it thin -- just re-export from `@/features/dashboard`:
```typescript

```
If the layout file imports from `-ui.navigation.tsx`, update that import to use `@/features/dashboard` directly.

Each route file should be under 20 lines after this conversion.
